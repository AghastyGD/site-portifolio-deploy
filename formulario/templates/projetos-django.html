Ecommerce Django + React

Introdução

Este projeto é um simples MiniShop desenvolvido com Django para o backend e React para o frontend, incluindo funcionalidades de pagamento usando a API de Braintree e Paypal. O aplicativo armazena informações de usuários, métodos de pagamento e pedidos em um banco de dados PostgreSQL.

Sistema de Login Completo

Introdução

Um Sistema de Login que permite que os usuários se registrem, façam login e realizem a autenticação em um ambiente web seguro. Utilizei o framework Django e algumas bibliotecas do Python como o SendGrid para fornecer recursos completos de autenticação e gerenciamento de usuários.
